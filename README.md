### Endpoints 
  * AddMovie:
    * { Title: string, Director: string, Duration: int, Genre: string }
  * GetMovies:
    * Returns a list of movies registered in the system
  * GetMoviesById: 
    * Returns a movie with the given ID
  * UpdateMovieById:
    * Updates a movie with the given ID
